# Yayasan Melati Rawasari Website

Website resmi untuk Yayasan Santunan Anak Yatim Piatu dan Dhuafa Melati Rawasari.

## Tentang Website

Website ini dibuat untuk memperkenalkan Yayasan Melati Rawasari dan kegiatan-kegiatan sosial yang mereka jalankan. Website ini dirancang dengan desain modern dan responsif yang dapat diakses melalui berbagai perangkat.

## Fitur

- **Desain Responsif**: Tampilan yang optimal di desktop, tablet, dan mobile
- **Navigasi Smooth**: Pengalaman navigasi yang halus antar section
- **Informasi Lengkap**: Berisi informasi tentang yayasan, kegiatan, dan cara berdonasi
- **Interaktif**: Fitur copy nomor rekening dengan satu klik
- **Animasi**: Animasi scroll yang menarik dan profesional

## Struktur File

```
├── index.html          # File HTML utama
├── styles.css          # File CSS untuk styling
├── script.js           # File JavaScript untuk interaktivitas
└── README.md           # Dokumentasi ini
```

## Seksi Website

### 1. Beranda (Hero Section)
- Pengenalan singkat tentang yayasan
- Call-to-action untuk donasi dan informasi lebih lanjut

### 2. Tentang Kami
- Sejarah dan misi yayasan
- Informasi legalitas
- Struktur pengurus

### 3. Kegiatan Kami
- Kegiatan keagamaan (Iqra, tahsin, murojaah)
- Santunan pendidikan
- Pembagian sembako
- Santunan hari raya
- Pemotongan hewan qurban

### 4. Donasi
- Informasi rekening donasi (Bank Syariah Indonesia & BCA)
- Fitur copy nomor rekening

### 5. Kontak
- Alamat lengkap
- Nomor telepon
- Link Google Maps

## Cara Menggunakan

1. **Buka Website**: Buka file `index.html` di browser web modern (Chrome, Firefox, Safari, Edge)

2. **Navigasi**: Gunakan menu navigasi untuk berpindah antar section
   - Klik menu untuk mobile (hamburger icon)
   - Scroll smooth untuk desktop

3. **Donasi**: 
   - Klik pada nomor rekening untuk menyalin
   - Notifikasi akan muncul saat nomor berhasil disalin

4. **Kontak**: Klik link Google Maps untuk melihat lokasi yayasan

## Teknologi yang Digunakan

- **HTML5**: Struktur konten semantik
- **CSS3**: Desain modern dengan animasi dan transisi
- **JavaScript (ES6+)**: Interaktivitas dan fitur dinamis
- **Google Fonts**: Typography profesional (Poppins)
